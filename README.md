I used a UCFK4 to implement a simple game using the navswitch, pushbutton and display. The game generates a random path from the centre of the display which the user must memorise and then recreate using the navswitch. If the player wins the round, the length of the path grows. A robust finite-state machine and task driven schedular were used to achieve this behaviour.
